The Host Role at Mallards

You are the first person every guest sees and the last person they interact with as they leave. Your role sets the tone for the entire experience.

Host Mission: Make every guest feel welcomed from the moment they walk in. Set accurate expectations, keep the floor moving, and stay calm during peak service.

Host Standards

  • Greet every guest within 30 seconds — even if you're on the phone or with another guest, acknowledge them with a nod and smile
  • Opening script: "Welcome to Mallards! How many are in your party today?"
  • If there's a wait: "We have about a [X]-minute wait. Can I get a name for your party?"
  • Never quote a low wait time just to make someone happy — honesty builds trust
  • Offer a menu while guests wait — it sets expectations and builds anticipation
  • Know the menu highlights well enough to answer common questions
  • Never say "I don't know" without following up with "Let me find out for you"

Phone Greeting Script

"Thank you for calling Mallards [Location], this is [Your Name] — how can I help you today?"

  • Speak clearly, smile while you talk (it comes through)
  • If placing on hold: ask first, return within 2 minutes
  • Reservations: get name, party size, date, time, contact number
  • Complaints: listen fully, apologize, take name/number, alert manager — never promise comps

Seating Rotation & Table Management

How Rotation Works

  • Rotate servers in order — keep sections balanced, prevent overload
  • Never skip a server without a reason
  • If a server is cut, remove them from rotation immediately
  • Always confirm the server is ready before bringing guests back

Special Seating Considerations

SituationWhat to Do
Guests with mobility needsSeat near entrance; accessible table

Large parties (6+)Confirm with manager and server before seating; ensure table is set
Window/patio requestsAccommodate when possible; communicate wait time honestly

When a Table Opens

Communicate it to the server and manager immediately. Every minute an open table sits empty is time on someone's wait.
